  3. class Emitter[T] extends Events[T] with Default[T] with EventSource with Reactor[T]

    An event stream value that can programatically emit events.

    An event stream value that can programatically emit events.

    Events are emitted to the event stream value by calling the += method. The emitter can be closed by calling the close method -- after this no more events will be accepted through +=.

    Example:

    val emitter = new Events.Emitter[Int]
val prints = emitter.onEvent(println)
emitter += 1
emitter += 2
